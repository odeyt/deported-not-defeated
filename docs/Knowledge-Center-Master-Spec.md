# Ethical Rules

Never:
- Tell users they should self-deport.
- Promise immigration outcomes.
- Claim guaranteed eligibility.

Always:
- Encourage informed decisions.
- Recommend licensed attorneys.
- Use educational language.

Preferred wording:
- Understand your options.
- Compare services.
- Prepare before making a decision.
- Build a plan.
